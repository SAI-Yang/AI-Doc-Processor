# AI 文档批处理工具

基于 AI 的文档批量处理桌面应用。支持拖拽文件、选择处理模板、预览结果、批量处理多格式文档。

## 功能特性

- **多格式支持**：DOCX、PDF、TXT、Markdown 格式的读取与处理
- **拖拽操作**：支持文件拖拽添加、文件夹批量添加
- **丰富模板**：内置中译英、英译中、学术润色、总结摘要、扩写等 9 种处理模板
- **流水线处理**：支持多模板串联的流水线处理模式
- **多 API 支持**：兼容 DeepSeek、OpenAI、Anthropic 及 OpenAI 兼容接口
- **实时预览**：左右分栏对比原文与处理结果，支持差异高亮
- **批量处理**：后台线程处理，支持暂停、继续、取消
- **处理历史**：自动记录处理任务，方便回溯
- **暗色主题**：现代化深色界面，护眼且美观

## 安装说明

### 环境要求

- Python 3.8+
- Windows 7+ / macOS 10.15+ / Linux

### 安装步骤

1. 克隆或下载项目到本地

2. 安装依赖：

```bash
cd ai-doc-processor
pip install -r requirements.txt
```

3. 可选依赖（用于 DOCX/PDF 格式支持）：

```bash
pip install python-docx pdfplumber
```

### 运行

```bash
python -m app.main
```

或

```bash
python app/main.py
```

### 打包为独立可执行文件

```bash
# 单文件模式
python build.py

# 目录模式
python build.py --dir

# 清理构建产物
python build.py --clean
```

## 使用指南

### 快速开始

1. **添加文件**：拖拽文件到左侧文件列表，或点击工具栏的"+ 添加文件"按钮
2. **选择模板**：在右侧模板面板点击选择一个处理模板（如"中译英"）
3. **配置参数**：调整温度和最大 Token 数滑块
4. **开始处理**：点击工具栏的"▶ 开始处理"按钮，或按 F5 键
5. **查看结果**：处理完成后在中央预览区域查看原文与结果对比
6. **导出结果**：点击"导出"按钮保存处理后的文件

### 模板说明

| 模板 | 说明 |
|------|------|
| 中译英 | 将中文文档翻译为英文，保留专业术语 |
| 英译中 | 将英文文档翻译为中文 |
| 学术润色 | 优化学术论文语法和表达 |
| 通用润色 | 修正语法错误，优化表达 |

| 要点提取 | 提取关键点和行动项 |

| 自定义 | 使用自定义提示词 |

### API 配置

在设置对话框（Ctrl+,）中配置 API：

1. **Provider**：选择 DeepSeek、OpenAI、Anthropic 或自定义
2. **API Key**：输入你的 API 密钥
3. **Base URL**：API 端点地址
4. **模型**：选择使用的模型
5. **测试连接**：点击按钮验证配置

### 流水线模式

1. 点击模板面板顶部的"流水线"按钮
2. 依次选择多个模板
3. 点击"+ 添加选中到流水线"按钮
4. 在流水线列表中拖拽调整顺序
5. 点击"开始处理"按顺序执行多个处理步骤

## 常见问题

**Q: 无法打开 DOCX/PDF 文件？**
A: 需要安装可选依赖：`pip install python-docx pdfplumber`

**Q: 处理速度慢怎么办？**
A: 在设置中增大"最大并发数"，或使用更快的 API 模型

**Q: 如何处理非常大的文件？**
A: 当前版本对长文本进行整体处理。建议先分割文档再批量处理

**Q: 自定义模板如何使用？**
A: 选择"自定义"模板，在下方输入系统提示词和用户提示词模板，
用户提示词中使用 `{text}` 表示文档内容位置

**Q: 如何备份配置和模板？**
A: 配置文件保存在 `~/.ai-doc-processor/` 目录，备份该目录即可

## 技术说明

- UI 框架：PyQt5
- 后端引擎：异步 LLM API 调用（支持流式输出）
- 文档解析：python-docx（DOCX）、pdfplumber（PDF）
- 打包工具：PyInstaller
- 配置存储：JSON 文件（~/.ai-doc-processor/）
